position property

static

element with static position can not move in all direction(top/left/right/bottom)

relative

element with relative position can move in all direction(top/left/right/bottom)

absoulte

element with absoulte position can move according to body in all direction(top/left/right/bottom)

absoulte with parent

fixed

element with fixed block set according to body .
it will be not scroll with other elements.

sticky

relative+fixed

intially work as a realtive

whemever its position true (Ex. tpo:10px)work as a fixed
